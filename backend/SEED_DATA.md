# Скрипт для заполнения тестовыми данными

Скрипт `scripts/seed_data.py` создает начальные данные для разработки и тестирования.

## Что создается

### 1. Языки
- Русский (ru)
- English (en)
- Español (es)

### 2. Категории заданий (12 категорий)
- Романтика (romance) - #FFC700
- Что-то новое (new) - #6049EC
- Эмоции (emotions) - #EB454E
- Время вместе (time) - #FFC700
- Игры и челенджы (games) - #FFC700
- Интим (intimate) - #EB454E
- Флирт (flirt) - #6049EC
- Творчество (creativity) - #FFC700
- Приключения (adventure) - #6049EC
- Путешествия (travel) - #FFC700
- Спорт (sport) - #EB454E
- Кулинария (cooking) - #6049EC

Все категории создаются с переводами на все три языка.

### 3. Примерные задания (5 заданий)
- Задания с переводами на ru, en, es
- Разные категории и целевые аудитории (male, female, couple, all)

### 4. Тестовый пользователь
- tg_id: 123456789
- username: test_user
- Имя: Тестовый Пользователь
- Пол: Пара (couple)
- Баланс: 100 искр
- Интересы: Романтика, Игры, Интим

## Запуск

### Из директории backend/
```bash
cd backend
python scripts/seed_data.py
```

### Из корневой директории проекта
```bash
python backend/scripts/seed_data.py
```

## Повторный запуск

Скрипт проверяет существующие данные и не создает дубликаты:
- Если язык уже существует - пропускает
- Если категория уже существует - пропускает, но добавляет недостающие переводы
- Если задание уже существует - создает новое (можно запускать несколько раз)
- Если тестовый пользователь уже существует - пропускает

## Важно

- Убедитесь, что миграции применены перед запуском скрипта
- Скрипт создает таблицы автоматически, если их нет
- Все данные создаются с `is_active=True`
