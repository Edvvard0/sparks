# Руководство по иконкам для Welcome страницы

## Структура папок для иконок

Все иконки должны быть сохранены в папке:
```
frontend/src/assets/icons/
```

## Необходимые иконки для Welcome страницы

### 1. Иконка молнии (Lightning Icon)
- **Путь:** `frontend/src/assets/icons/lightning-icon_welcome.svg`
- **Использование:** Кнопка "ЗАДАНИЯ ДЛЯ ПАР"
- **Размер:** 20x20px в интерфейсе
- **Статус:** ✅ Обновлена - используется `lightning-icon_welcome.svg`

### 2. Иконка кубика (Dice/Games Icon)
- **Путь:** `frontend/src/assets/icons/games-icon_welcome.svg`
- **Использование:** Кнопка "Миниигры"
- **Размер:** 20x20px в интерфейсе
- **Описание:** Иконка должна быть в виде кубика (dice) с точками на гранях
- **Статус:** ✅ Обновлена - используется `games-icon_welcome.svg`

### 5. Фоновый логотип (Logo Background)
- **Путь:** `frontend/src/assets/icons/Logo.svg`
- **Использование:** Фоновое изображение на странице Welcome
- **Размер:** 493x499px (масштабируется)
- **Описание:** Красный фон с органическими узорами (стилизованное сердце)
- **Статус:** ✅ Используется как фон страницы Welcome

### 3. Иконка для "ДЛЯ НЕЁ" (опционально)
- **Путь:** `frontend/src/assets/icons/female-icon.svg`
- **Использование:** Можно добавить иконку к кнопке "ДЛЯ НЕЁ"
- **Размер:** 20x20px в интерфейсе
- **Статус:** ✅ Уже существует (не используется в текущей версии)

### 4. Иконка для "ДЛЯ НЕГО" (опционально)
- **Путь:** `frontend/src/assets/icons/male-icon.svg`
- **Использование:** Можно добавить иконку к кнопке "ДЛЯ НЕГО"
- **Размер:** 20x20px в интерфейсе
- **Статус:** ✅ Уже существует (не используется в текущей версии)

## Требования к иконкам

1. **Формат:** SVG (предпочтительно) или PNG с прозрачным фоном
2. **Цвет:** Белый (#FFFFFF) для отображения на темном фоне
3. **Размер:** Оптимальный размер для отображения 20x20px в интерфейсе
4. **Стиль:** Простые, минималистичные иконки, соответствующие дизайну приложения

## Проверка иконок

Проверьте, что `games-icon.svg` действительно является иконкой кубика. Если это не так, замените её на правильную иконку кубика.

## Использование в коде

Иконки импортируются в компоненте `Welcome.tsx`:

```typescript
import lightningIcon from '../assets/icons/lightning-icon_welcome.svg'
import gamesIcon from '../assets/icons/games-icon_welcome.svg'
import logoBg from '../assets/icons/Logo.svg'
```

И используются в кнопках и как фон:

```tsx
{/* Фоновый логотип */}
<img src={logoBg} alt="Logo Background" className="w-full h-full object-contain" />

{/* Иконки в кнопках */}
<img src={lightningIcon} alt="Lightning" width={20} height={20} className="flex-shrink-0" />
<img src={gamesIcon} alt="Games" width={20} height={20} className="flex-shrink-0" />
```

## Расположение кнопок

Все кнопки на странице Welcome расположены **вертикально** (одна под другой):
1. "ДЛЯ НЕЁ" (верхняя)
2. "ЗАДАНИЯ ДЛЯ ПАР" (с иконкой молнии)
3. "ДЛЯ НЕГО"
4. "Миниигры" (с иконкой кубика)
5. "Поехали" (темная кнопка внизу)
