# Sparks - Telegram Mini App

React приложение с TypeScript и Tailwind CSS для Telegram Mini App.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Компоненты

### Onboarding
Компонент онбординга с выбором пола и направлений интересов.

## Технологии

- React 18+
- TypeScript
- Tailwind CSS
